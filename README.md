# Multi-Agent Competitive Analysis

Parallel competitive analysis of AI coding assistant orchestration tools,
executed by Gas Town's multi-polecat workflow.

## Scope

Comparing **Gas Town** against:
- Cursor Composer
- Windsurf Cascade
- Devin (Cognition)
- OpenHands (formerly OpenDevin)
- SWE-agent (Princeton NLP)

| # | Report | File |

| 3 | Pricing & Business Model | `reports/03-pricing-business.md` |
| 4 | Community & Developer Sentiment | `reports/04-community-sentiment.md` |
| 5 | SWOT Analysis | `reports/05-swot-analysis.md` |
| 6 | **Final Synthesis** | `final/compete-report.md` |

## How This Was Made

This analysis was produced by 6 AI agents (polecats) coordinated by a Mayor agent
in Gas Town. Wave 1 (reports 1-5) ran in parallel; Wave 2 (report 6) synthesized
all findings into the final deliverable.
